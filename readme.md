
# Dash Chat Room Over Cache

A chat room application built with Dash framework, leveraging caching mechanisms for efficient message storage and retrieval.

## Project Structure

```
pages/
├── loggin.py      # User authentication and login functionality
├── room.py        # Chat room main interface and messaging logic
├── welcome.py     # Welcome/landing page
└── __pycache__/   # Python cache files
```

## Features

- **User Authentication**: Login system for user access
- **Chat Room**: Real-time messaging interface
- **Cache Optimization**: Efficient data management using caching

## Installation

```bash
pip install dash
```

## Usage

Run the application:

```bash
python app.py
```
